# یادسپار v2

PWA کاملاً لوکال برای:
- لغات انگلیسی: بازیابی فعال + مرور فاصله‌دار
- شماره موبایل: chunked recall
- لیست‌های کوتاه: prospective reminders

## تغییرات این نسخه
- تم شب / روشن / سیستم
- داشبورد خلوت‌تر با آیکون، ایموجی و حلقه‌های پیشرفت
- توضیحات اضافی پشت دکمه `i`
- ورودی هوشمند انعطاف‌پذیرتر:
  - ترکیب انگلیسی + فارسی → لغت
  - ترکیب نام + شماره → شماره
  - ترکیب اقلام / خط‌ها / ویرگول‌ها → لیست
  - چند بلوک و چند خط به‌صورت ترکیبی نیز تحلیل می‌شوند

## اجرا
1. این پوشه را روی یک static server اجرا کنید.
2. روی آیفون با Safari باز کنید.
3. Share → Add to Home Screen.

## گزینه‌های ساده برای اجرا
- Python: `python3 -m http.server 8080`
- Node: هر static server
